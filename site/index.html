Why Aelix

Extensions are just Python

A tool is a plain function вЂ” no plugin language, no out-of-process bridge. Import pandas, an internal SDK, or a warehouse client and hand results straight back to the model, in-process. Hot-reload with /reload, no restart.

Runs on budgets you already own

Hand-written native adapters for Anthropic, OpenAI, Gemini/Vertex, OpenRouter, Cloudflare, and GitHub Copilot вЂ” including the seat you already sign in with. Route cheap work and hard reasoning to different models from one session.

Signed, air-gap-ready trust

Ed25519 provenance and SHA-256 pinning on extensions (keygen | sign | trust, fail-closed --require-signature), an offline catalog, and an --offline mode built for closed networks. Policy and guardrails are auditable built-in extensions.
