## §0 — W0 INVESTIGATION FINDINGS (Pi drift verified at SHA 734e08e)

### P-163 — Pi `models.generated.ts` is 428 KB / ~10,500 lines — verbatim port infeasible in one sprint

W0 measurement: `packages/ai/src/models.generated.ts` = **427,821 bytes** (auto-generated by Pi). Mirroring as a Python module would be a single 400+ KB file. Sprint 6f₁ scopes to RUNTIME (the `ModelRegistry` class + 7 helpers), with a **~12-model seed catalog** so the 3 RPC commands have something to enumerate. Sprint 6g does the data-transfer port.

**Decision:** Aelix `aelix_ai.models_generated` ships as Python dict literal with ~12 models across 3 providers (Anthropic + OpenAI + OpenRouter). Sprint 6g converts the full 428KB Pi TS file → Python equivalent (or JSON data file loaded at module import).

### P-164 — Pi `ai/src/models.ts` is 92 LOC pure helpers, no AuthStorage dependency

Pi `models.ts:1-92`:
- `getModel(provider, modelId)` — `modelRegistry.get(provider)?.get(modelId)`
- `getProviders()` — `Array.from(modelRegistry.keys())`
- `getModels(provider)` — `Array.from(modelRegistry.get(provider)?.values() ?? [])`
- `calculateCost(model, usage)` — `(cost.X / 1_000_000) * usage.X` for input/output/cacheRead/cacheWrite + total
- `getSupportedThinkingLevels(model)` — filter `EXTENDED_THINKING_LEVELS` against `model.thinkingLevelMap`
- `clampThinkingLevel(model, level)` — clamp request to nearest supported level (scan forward then backward)
- `modelsAreEqual(a, b)` — compare `id` + `provider`

The `modelRegistry: Map<string, Map<string, Model>>` is constructed at module load from `MODELS` constant in `models.generated.ts`. Aelix port: a module-level `_PROVIDER_MODELS: dict[str, dict[str, Model]]` initialized at import time from `models_generated.MODELS`.

**Decision:** Port `models.ts` verbatim to `packages/aelix-ai/src/aelix_ai/models.py`. The existing `aelix_ai.models.clamp_thinking_level` helper (Sprint 6b stub) is REPLACED by the full Pi-parity port.

### P-165 — Pi `coding-agent/src/core/model-registry.ts` ModelRegistry class (820 LOC) has 14 public methods + AuthStorage + OAuth integration

Pi exposes:
- Factory: `static create(authStorage, modelsJsonPath?)`, `static inMemory(authStorage)`
- Model access: `getAll()`, `getAvailable()`, `find(provider, modelId)`
- Auth resolution: `hasConfiguredAuth(model)`, `getApiKeyAndHeaders(model)`, `getApiKeyForProvider(provider)`, `getProviderAuthStatus(provider)`, `isUsingOAuth(model)`
- Lifecycle: `refresh()`, `getError()`
- Dynamic registration: `registerProvider(name, config)`, `unregisterProvider(name)`
- Display: `getProviderDisplayName(provider)`

The constructor takes an `AuthStorage` (Sprint 6c) + optional `modelsJsonPath`. The `getApiKeyAndHeaders(model)` returns `ResolvedRequestAuth = {ok: true, apiKey?, headers?} | {ok: false, error: string}` and is the canonical auth resolver that bridges ModelRegistry to the stream adapter's `get_api_key_and_headers` callback (Sprint 6a contract).

**Decision:** Aelix ports 14 methods. The `loadModels()` pipeline (load builtins + load custom from models.json + merge + OAuth modifyModels) is implemented with a placeholder for models.json (Sprint 6g ports the schema validator). The `modifyModels` callback (Sprint 6e P-132) is wired through `refresh()`.

### P-166 — Pi ModelRegistry depends on `models.json` for custom-model overrides

Pi `model-registry.ts:loadCustomModels(path)` reads a user-editable `models.json` file with:
- Comment + trailing-comma stripping
- TypeBox schema validation (`ProviderConfigSchema`, `ModelDefinitionSchema`)
- Per-provider config overrides (apiKey / headers / authHeader / oauth / models)

**Decision:** Sprint 6f₁ DEFERS models.json loading — `loadModels()` only loads built-ins (from `aelix_ai.models_generated.MODELS`) + applies OAuth `modify_models` callbacks. `modelsJsonPath` parameter accepted for API parity but raises `NotImplementedError` if non-None until Sprint 6g.

### P-167 — Aelix `streaming.Model` dataclass missing 5 Pi fields critical for ModelRegistry

Pi `Model` type (verified against `ai/src/types.ts`):
- `id: string`
- `name: string`
- `provider: string`
- `api: Api`
- `cost: {input, output, cacheRead, cacheWrite}` (per million tokens)
- `thinkingLevelMap?: dict[level, value | null]`
- `maxTokens: number`
- `contextWindow: number`
- `knowledgeCutoff?: string`
- `releaseDate?: string`
- `reasoning?: boolean`
- `baseUrl?: string`
- `headers?: dict`
- `input: string[]` (e.g., `["text", "image"]`)
- `compat?: OpenAICompletionsCompat | ...` (per-API compat overrides)

Aelix `streaming.Model` currently has: `id`, `name`, `provider`, `api`, `reasoning`, `base_url`, `headers`, `input`.

**Missing for Sprint 6f₁:**
- `cost: ModelCost` (NEW dataclass with 4 fields per million)
- `thinking_level_map: dict[str, str | int | None] | None`
- `max_tokens: int`
- `context_window: int`

**Decision:** Extend `streaming.Model` with 4 new fields as ADDITIVE (defaults = sentinel/0/None) so Sprint 6a/6b/6c/6d/6e code is unaffected. The seed catalog populates these for the 12 seed models. Document in ADR-0064.

### P-168 — RPC `set_model` requires harness mutable `current_model` property

Pi `session.setModel(provider, modelId)` updates session state and emits a session event. Aelix's `AgentHarness` currently has `model` as a constructor-time field (Sprint 6a). Sprint 6f needs:
- Public setter `harness.set_current_model(model: Model)` 
- Public property `harness.current_model: Model | None`
- The RPC `set_model` handler calls `registry.find(provider, modelId)` then `harness.set_current_model(model)` and returns the Model in the response

**Decision:** Add public `current_model` property + setter to `AgentHarness`. The current `model` field becomes the "initial model"; `current_model` is the runtime-mutable state. RPC `get_state` already reads model — wire it to `current_model` if non-None, else fall back to `model`.

### P-169 — `cycle_model` rotates in the order returned by `getAvailable()`

Pi `session.cycleModel()` rotates to the next model in `modelRegistry.getAvailable()` (alphabetical-by-(provider, id) per Pi `model-registry.ts:getAvailable`). The response includes `{model, thinkingLevel, isScoped}` where `isScoped` indicates the model was scoped to a workspace config (Sprint 6g).

**Decision:** Sprint 6f₁ `cycle_model` uses `getAvailable()` ordering, returns the new model + clamped thinking level + `isScoped: false` (always — workspace scoping is Sprint 6g). If no models are available, returns `data: None` per Pi.

## §C — `aelix_ai/models.py` REWRITE

Replace the Sprint 6b stub with full Pi-parity port:

```python
"""Pi parity: ``packages/ai/src/models.ts`` (SHA 734e08e).

Provider→model catalog accessor + cost calculator + thinking-level clamp.
"""

from typing import Any
from aelix_ai.models_generated import MODELS
from aelix_ai.streaming import Model, Usage  # Usage TBD or inline

# Module-load: build provider→{model_id: Model} index from MODELS dict.
_PROVIDER_MODELS: dict[str, dict[str, Model]] = {}
for provider_name, models_dict in MODELS.items():
    _PROVIDER_MODELS[provider_name] = {}
    for model_id, model in models_dict.items():
        _PROVIDER_MODELS[provider_name][model_id] = model


def get_model(provider: str, model_id: str) -> Model | None:
    """Pi parity: ``models.ts:20-26``."""
    return _PROVIDER_MODELS.get(provider, {}).get(model_id)


def get_providers() -> list[str]:
    """Pi parity: ``models.ts:28-30``."""
    return list(_PROVIDER_MODELS.keys())


def get_models(provider: str) -> list[Model]:
    """Pi parity: ``models.ts:32-37``."""
    return list(_PROVIDER_MODELS.get(provider, {}).values())


EXTENDED_THINKING_LEVELS: list[str] = ["off", "minimal", "low", "medium", "high", "xhigh"]
"""Pi parity: ``models.ts:48``."""


def calculate_cost(model: Model, usage: Usage) -> dict[str, float]:
    """Pi parity: ``models.ts:39-46``.

    Mutates ``usage.cost`` in-place (matches Pi behavior).
    """
    usage.cost.input = (model.cost.input / 1_000_000) * usage.input
    usage.cost.output = (model.cost.output / 1_000_000) * usage.output
    usage.cost.cache_read = (model.cost.cache_read / 1_000_000) * usage.cache_read
    usage.cost.cache_write = (model.cost.cache_write / 1_000_000) * usage.cache_write
    usage.cost.total = (
        usage.cost.input + usage.cost.output + usage.cost.cache_read + usage.cost.cache_write
    )
    return usage.cost


def get_supported_thinking_levels(model: Model) -> list[str]:
    """Pi parity: ``models.ts:50-59``."""
    if not model.reasoning:
        return ["off"]
    out: list[str] = []
    for level in EXTENDED_THINKING_LEVELS:
        mapped = (model.thinking_level_map or {}).get(level)
        if mapped is None and level != "off":
            # Pi: null means NOT supported
            continue
        if level == "xhigh" and mapped is None:
            continue
        out.append(level)
    return out


def clamp_thinking_level(model: Model, level: str) -> str:
    """Pi parity: ``models.ts:61-80``."""
    available = get_supported_thinking_levels(model)
    if level in available:
        return level
    if level not in EXTENDED_THINKING_LEVELS:
        return available[0] if available else "off"
    idx = EXTENDED_THINKING_LEVELS.index(level)
    for i in range(idx, len(EXTENDED_THINKING_LEVELS)):
        cand = EXTENDED_THINKING_LEVELS[i]
        if cand in available:
            return cand
    for i in range(idx - 1, -1, -1):
        cand = EXTENDED_THINKING_LEVELS[i]
        if cand in available:
            return cand
    return available[0] if available else "off"


def models_are_equal(a: Model | None, b: Model | None) -> bool:
    """Pi parity: ``models.ts:86-92``."""
    if a is None or b is None:
        return False
    return a.id == b.id and a.provider == b.provider
```

Note: `Usage` may already be defined; if not, add a minimal Usage dataclass.

## §E — `aelix_coding_agent/model_registry.py` (NEW)

Port Pi `coding-agent/src/core/model-registry.ts` runtime (subset).

```python
"""ModelRegistry — Pi parity: ``coding-agent/src/core/model-registry.ts``.

Sprint 6f₁ ships the RUNTIME (14 methods). Sprint 6g ports the
``models.json`` schema validator + full catalog + ``model-resolver.ts``.
"""

from dataclasses import dataclass, field
from typing import Any
from aelix_ai.models import get_providers, get_models
from aelix_ai.oauth import AuthStorage, AuthStatus, get_oauth_provider
from aelix_ai.streaming import Model


@dataclass
class ResolvedRequestAuth:
    """Pi parity: ``model-registry.ts::ResolvedRequestAuth``."""
    ok: bool
    api_key: str | None = None
    headers: dict[str, str] = field(default_factory=dict)
    error: str | None = None


@dataclass
class ProviderConfigInput:
    """Pi parity: ``model-registry.ts::ProviderConfigInput`` (subset)."""
    api_key: str | None = None
    headers: dict[str, str] | None = None
    auth_header: str | None = None
    oauth: Any | None = None  # OAuthProvider — Sprint 6g wires full schema
    models: dict[str, Model] | None = None  # custom catalog entries


class ModelRegistry:
    """Pi parity: ``coding-agent/src/core/model-registry.ts:ModelRegistry`` (subset)."""

    def __init__(
        self,
        auth_storage: AuthStorage,
        models_json_path: str | None = None,
    ) -> None:
        if models_json_path is not None:
            raise NotImplementedError(
                "models.json loading deferred to Sprint 6g (ADR-0066 carry-forward)"
            )
        self._auth_storage = auth_storage
        self._models_json_path = models_json_path
        self._models: list[Model] = []
        self._provider_request_configs: dict[str, ProviderConfigInput] = {}
        self._registered_providers: dict[str, ProviderConfigInput] = {}
        self._load_error: str | None = None
        self._load_models()

    @classmethod
    def create(
        cls, auth_storage: AuthStorage, models_json_path: str | None = None,
    ) -> "ModelRegistry":
        """Pi parity: ``model-registry.ts::ModelRegistry.create``."""
        return cls(auth_storage, models_json_path)

    @classmethod
    def in_memory(cls, auth_storage: AuthStorage) -> "ModelRegistry":
        """Pi parity: ``model-registry.ts::ModelRegistry.inMemory``."""
        return cls(auth_storage, None)

    # ── Model access ───────────────────────────────────────────────
    def get_all(self) -> list[Model]:
        """Pi parity: ``model-registry.ts::getAll``."""
        return list(self._models)

    def get_available(self) -> list[Model]:
        """Pi parity: ``model-registry.ts::getAvailable``.

        Filters to models for which ``has_configured_auth(model)`` is True.
        """
        return [m for m in self._models if self.has_configured_auth(m)]

    def find(self, provider: str, model_id: str) -> Model | None:
        """Pi parity: ``model-registry.ts::find``."""
        for m in self._models:
            if m.provider == provider and m.id == model_id:
                return m
        return None

    # ── Auth resolution ────────────────────────────────────────────
    def has_configured_auth(self, model: Model) -> bool: ...
    async def get_api_key_and_headers(self, model: Model) -> ResolvedRequestAuth: ...
    async def get_api_key_for_provider(self, provider: str) -> str | None: ...
    async def get_provider_auth_status(self, provider: str) -> AuthStatus: ...
    def is_using_oauth(self, model: Model) -> bool: ...

    # ── Lifecycle ──────────────────────────────────────────────────
    def refresh(self) -> None:
        """Pi parity: ``model-registry.ts::refresh``.

        Reloads models from disk + rebuilds dynamic OAuth/API
        registrations + applies OAuth modify_models callbacks.
        """
        self._load_models()

    def get_error(self) -> str | None:
        return self._load_error

    # ── Dynamic registration ───────────────────────────────────────
    def register_provider(self, name: str, config: ProviderConfigInput) -> None: ...
    def unregister_provider(self, name: str) -> None: ...

    # ── Display ────────────────────────────────────────────────────
    def get_provider_display_name(self, provider: str) -> str: ...

    # ── Loading pipeline ───────────────────────────────────────────
    def _load_models(self) -> None:
        """Pi parity: ``model-registry.ts::loadModels``."""
        # 1. Load built-ins from aelix_ai.models
        builtins: list[Model] = []
        for provider in get_providers():
            builtins.extend(get_models(provider))
        # 2. Sprint 6g: load custom from models.json
        # 3. Apply OAuth modify_models callbacks per registered OAuth provider
        for oauth_provider in self._auth_storage_oauth_providers():
            modify = getattr(oauth_provider, "modify_models", None)
            if modify and callable(modify):
                creds = self._auth_storage.get_oauth_sync(oauth_provider.id)
                if creds:
                    builtins = modify(builtins, creds)
        self._models = builtins
```

The `get_api_key_and_headers(model)` bridges to `AuthStorage.get_api_key_cascade(provider, include_fallback=False)` per Sprint 6e cascade contract.

### G.2 `_handle_cycle_model`

```python
async def _handle_cycle_model(harness, registry, cmd):
    available = registry.get_available()
    if not available:
        return RpcSuccessResponse(command="cycle_model", data=None)
    current = harness.current_model
    next_index = 0
    if current is not None:
        for i, m in enumerate(available):
            if models_are_equal(m, current):
                next_index = (i + 1) % len(available)
                break
    next_model = available[next_index]
    harness.set_current_model(next_model)
    thinking_level = clamp_thinking_level(next_model, harness.thinking_level)
    return RpcSuccessResponse(
        command="cycle_model",
        data={
            "model": _model_to_dict(next_model),
            "thinkingLevel": thinking_level,
            "isScoped": False,  # Sprint 6g
        },
    )
```

### G.3 `_handle_get_available_models`

```python
async def _handle_get_available_models(harness, registry, cmd):
    available = registry.get_available()
    return RpcSuccessResponse(
        command="get_available_models",
        data={"models": [_model_to_dict(m) for m in available]},
    )
```
