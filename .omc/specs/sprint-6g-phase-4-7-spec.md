### P-200 — Pi `model.compat` field is per-API discriminated union; Aelix Sprint 6g₁ stores as `dict[str, Any] | None`

Sprint 6g₂ types `Model.compat` as a discriminated union and wires the adapter to read it.

## §D — `aelix_ai/models_generated.py` REWRITE

Replace the 13-seed dict with a JSON loader:

```python
"""Pi parity: ``packages/ai/src/models.generated.ts`` (SHA 734e08e).

Sprint 6g₁ ships the FULL 32-provider catalog (~200+ models) loaded
from ``models_generated.json`` at module import. Replaces the Sprint
6f₁ 13-model seed.
"""

from __future__ import annotations
import json
from pathlib import Path
from typing import Any
from aelix_ai.streaming import Model, ModelCost

_CATALOG_JSON_PATH: Path = Path(__file__).parent / "models_generated.json"


def _load_catalog() -> dict[str, dict[str, Model]]:
    """Load the Pi catalog at module import.

    Pi camelCase keys are translated to snake_case Aelix Model fields.
    Order of providers and models follows JSON insertion order (Python
    3.7+ guarantees dict order preservation).
    """
    with _CATALOG_JSON_PATH.open(encoding="utf-8") as f:
        raw: dict[str, dict[str, dict[str, Any]]] = json.load(f)

    catalog: dict[str, dict[str, Model]] = {}
    for provider_name, models_dict in raw.items():
        catalog[provider_name] = {}
        for model_id, entry in models_dict.items():
            cost_dict = entry.get("cost") or {}
            cost = ModelCost(
                input=float(cost_dict.get("input", 0.0)),
                output=float(cost_dict.get("output", 0.0)),
                cache_read=float(cost_dict.get("cacheRead", 0.0)),
                cache_write=float(cost_dict.get("cacheWrite", 0.0)),
            )
            model = Model(
                id=entry["id"],
                name=entry.get("name", entry["id"]),
                api=entry["api"],
                provider=entry["provider"],
                base_url=entry.get("baseUrl", ""),
                reasoning=bool(entry.get("reasoning", False)),
                input=list(entry.get("input", [])),
                cost=cost,
                context_window=int(entry.get("contextWindow", 0)),
                max_tokens=int(entry.get("maxTokens", 0)),
                thinking_level_map=entry.get("thinkingLevelMap"),
                headers=entry.get("headers"),
                compat=entry.get("compat"),
            )
            catalog[provider_name][model_id] = model
    return catalog


MODELS: dict[str, dict[str, Model]] = _load_catalog()
"""Pi parity: ``MODELS`` constant. Loaded at import from JSON catalog."""


__all__ = ["MODELS"]
```

### F.1 Glob matching (Pi `:225-227`)

Pi uses `minimatch(fullId, globPattern, {nocase: true})`. Python equivalent: `fnmatch.fnmatchcase(haystack.casefold(), pattern.casefold())`. Note: `fnmatch.fnmatch` is case-insensitive on Windows by default — use `fnmatchcase` with explicit case-fold for cross-platform parity.

### F.2 `chalk` color output

Pi `console.warn(chalk.yellow(...))` becomes `print(f"Warning: ...", file=sys.stderr)` in Python (no color). Sprint 6h/Phase 5 wires chalk-equivalent via `rich`.

### F.3 Sort order

Pi `.sort((a, b) => b.id.localeCompare(a.id))` (descending) → Python `sorted(matches, key=lambda m: m.id, reverse=True)`.
